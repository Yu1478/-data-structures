## 1. 小鱼的数字游戏

**分值：** 10 分

**题目描述**  
小鱼最近被要求参加一个数字游戏，把看到的一串数字 `a_i`（长度不一定，以 `0` 结束）记住然后反着念出来（结束数字 `0` 不念）。请编程帮小鱼实现。

**输入描述**  
一行内输入一串整数，以 `0` 结束，以空格间隔。

**输出描述**  
一行内倒着输出这一串整数，以空格间隔。

**输入输出样例**

**输入**

```
3 65 23 5 34 1 30 0
```

**输出**

```
30 1 34 5 23 65 3
```

```c
#include <stdio.h>

int main() {
    int a[105], n = 0;
    int x;

    while (scanf("%d", &x) && x != 0) {
        a[n++] = x;
    }

    for (int i = n - 1; i >= 0; i--) {
        printf("%d", a[i]);
        if (i > 0) printf(" ");
    }

    printf("\n");
    return 0;
}
```

## 2. 小鱼比可爱

**分值：** 10 分

**题目描述**  
小鱼比赛可爱程度，每只鱼只能看到左边的鱼。计算每只鱼眼中有多少鱼不如自己可爱。

**输入描述**

```
第一行：正整数 n，表示鱼的数目
第二行：n 个整数，表示每只鱼的可爱程度 a_i
```

**输出描述**  
一行 `n` 个整数，每个整数表示该鱼眼中不如自己可爱的鱼的数量。

**输入输出样例**

**输入**

```
6
4 3 0 5 1 2
```

**输出**

```
0 0 0 3 1 2
```

```c
#include <stdio.h>

int comparefish(int* a, int n) {
    int total = 0;
    for (int i = 0; i < n; i++) {
        if (a[i] < a[n]) total++;
    }
    return total;
}

int main() {
    int n, a[100];
    scanf("%d", &n);

    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }

    for (int i = 0; i < n; i++) {
        printf("%d ", comparefish(a, i));
    }
    printf("\n");

    return 0;
}
```

## 3. 杨辉三角

**分值：** 10 分

**题目描述**  
输出杨辉三角的前 `n` 行。

**输入描述**  
输入一个数字 `n`（`n ≤ 20`）。

**输出描述**  
输出杨辉三角前 `n` 行。

**输入输出样例**

**输入**

```
6
```

**输出**

```
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
1 5 10 10 5 1
```

```c
#include <stdio.h>

int YHtriangle(int n, int m) {
    if (n == 1 || m == 1 || m == n) return 1;
    return YHtriangle(n - 1, m - 1) + YHtriangle(n - 1, m);
}

int main() {
    int n;
    scanf("%d", &n);

    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            printf("%d", YHtriangle(i, j));
            if (j < i) printf(" ");
        }
        printf("\n");
    }

    return 0;
}
```

## 4. 冰雹猜想

**分值：** 10 分

**题目描述**  
给定正整数 `n`，进行如下操作直到 `1`：

- 奇数 → `3n + 1`
- 偶数 → `n / 2`

输出从 `1` 开始倒序的变化数列。

**输入描述**  
输入一个正整数 `n`。

**输出描述**  
输出倒序变化数列，空格隔开。

**输入输出样例**

**输入**

```
20
```

**输出**

```
1 2 4 8 16 5 10 20
```

```c
#include <stdio.h>

int operation(int n) {
    return (n % 2 == 0) ? n / 2 : 3 * n + 1;
}

int main() {
    int a[1000], n, i = 0;
    scanf("%d", &n);

    while (n != 1) {
        a[i++] = n;
        n = operation(n);
    }
    a[i++] = 1;

    for (i--; i >= 0; i--) {
        printf("%d ", a[i]);
    }
    printf("\n");

    return 0;
}
```

## 5. 液晶屏

**分值：** 10 分

**题目描述**  
给出数字位数和数字串，输出 `3 × 5` 点阵液晶显示效果，数字间用 `.` 间隔。

**输入描述**

```
第一行：数字位数 n
第二行：长度为 n 的数字串
```

**输出描述**  
输出 `5` 行液晶屏显示。

**输入输出样例**

**输入**

```
10
0123456789
```

**输出**

```
XXX...X.XXX.XXX.X.X.XXX.XXX.XXX.XXX.XXX
X.X...X...X...X.X.X.X...X.....X.X.X.X.X
X.X...X.XXX.XXX.XXX.XXX.XXX...X.XXX.XXX
X.X...X.X.....X...X...X.X.X...X.X.X...X
XXX...X.XXX.XXX...X.XXX.XXX...X.XXX.XXX
```

```c
#include <stdio.h>

int main() {
    int n;
    char input[105];

    char shape[10][5][4] = {
        {"XXX", "X.X", "X.X", "X.X", "XXX"},
        {"..X", "..X", "..X", "..X", "..X"},
        {"XXX", "..X", "XXX", "X..", "XXX"},
        {"XXX", "..X", "XXX", "..X", "XXX"},
        {"X.X", "X.X", "XXX", "..X", "..X"},
        {"XXX", "X..", "XXX", "..X", "XXX"},
        {"XXX", "X..", "XXX", "X.X", "XXX"},
        {"XXX", "..X", "..X", "..X", "..X"},
        {"XXX", "X.X", "XXX", "X.X", "XXX"},
        {"XXX", "X.X", "XXX", "..X", "XXX"}
    };

    scanf("%d", &n);
    scanf("%s", input);

    for (int row = 0; row < 5; row++) {
        for (int i = 0; i < n; i++) {
            int num = input[i] - '0';
            printf("%s", shape[num][row]);
            if (i < n - 1) printf(".");
        }
        printf("\n");
    }

    return 0;
}
```

## 6. N 皇后

**分值：** 10 分

**题目描述**  
`n` 皇后问题：在 `n × n` 的棋盘上放置 `n` 个皇后，使皇后之间不能互相攻击（即任意两个皇后不在同一行、同一列或同一对角线上），输出解法总数。

**输入描述**  
输入一个整数 `n`。

**输出描述**  
输出方案总数。

**输入输出样例**

**输入**

```
4
```

**输出**

```
2
```

```c
#include <stdio.h>

int n, count = 0;
int cols[20] = {0};
int d1[40] = {0};
int d2[40] = {0};

void backtrack(int row) {
    if (row == n) {
        count++;
        return;
    }

    for (int col = 0; col < n; col++) {
        if (!cols[col] && !d1[row + col] && !d2[row - col + n]) {
            cols[col] = d1[row + col] = d2[row - col + n] = 1;
            backtrack(row + 1);
            cols[col] = d1[row + col] = d2[row - col + n] = 0;
        }
    }
}

int main() {
    scanf("%d", &n);
    backtrack(0);
    printf("%d\n", count);
    return 0;
}
```
